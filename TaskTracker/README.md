# Task Tracker

Used Angular, TypeScript, JSON Server, RxJS, Router, Font Awesome etc.

## Serve

### JSON server

```
npm run server
or
json-server --watch db.json --port 5000
```

### Task Tracker App

```
npm start
or
ng serve
```
